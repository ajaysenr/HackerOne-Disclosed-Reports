## Program Summary

> Please note, since this report was submitted and resolved, Oberlo has been removed from scope for our program. **We are no longer accepting reports on Oberlo at the time of this disclosure.**

Oberlo was an application that integrated with the Shopify platform. It was designed to simplify the process of sourcing and importing products from various suppliers into a Shopify store.

Due to missing access control checks on the backend, a user with limited permissions on an Oberlo account was able to modify the Oberlo subscription for the associated account.

## Vulnerability Details

## Summary:

Within Oberlo, it's possible to have a bare permission user with only access to the dashboard. This user can make an API call which will cancel the subscription.
