## Program Summary

> The researcher reported a reflected XSS in the Help Center AI assistant greeting flow. A cross-site POST to the Help Center search route could set an attacker-controlled `greeting` value, which was then rendered as markdown in the assistant greeting. By using a markdown image or link target with a `javascript:` URL, the attacker could execute JavaScript in an authenticated `help.shopify.com` session after the victim opened the crafted page and activated the rendered greeting link.

Shopify fixed the issue by removing the attacker-controlled greeting input path. This report was assessed in 2024 under the version of the Shopify Bug Bounty Calculator in use at the time. The report demonstrated a concrete, reproducible issue in a Shopify-owned authenticated application outside storefront execution. This is the kind of concise, high-quality research we have come to expect from this researcher, and it provides a strong example of an in-scope XSS report.

## Summary
I was able to leverage a potential CSRF at `help.shopify.com` which can lead to RXSS via the AI assistant chat greetings by abusing the way markdown images are rendered in the chat.

The AI bot at `help.shopify.com` has the ability to use markdown to improves conversation experience, while we customer, can't really use it. In the greetings process, I noticed that the URL was being reflected which lead me to investigate and see if it could be abused in some ways.

## Steps To Reproduce
1. Browse to `https://help.shopify.com` and log in so that the described behavior work


2. Visit the following POC https://████████/shopify-help-xss which contains the initial CSRF and setup the attack


3. You should be automatically redirected to `https://help.shopify.com/en/search/{some-token-here}` and the AI chat greetings should contain the broken markdown image:
    ████████


4. Because the link has `target=_blank`, you will sadly need to mouse wheel click the link to trigger the XSS. It should fire both graphql request which you can observe in your dev console network tabs


5. First, notice that some of your PII have been leaked to `https://████████/`


6. Second, notice that `saltymermaid@wearehackerone.com` was added as a subscriber of your last conversation
