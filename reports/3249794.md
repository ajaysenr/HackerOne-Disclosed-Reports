## Program Summary

> Title: SQL Injection in Structure Plugin
Vulnerability: An SQL injection flaw was discovered in ExpressionEngine's Structure plugin, specifically in the delete_channels() function. User input from the channel_ids parameter is passed directly into SQL queries without proper sanitization. Although the vulnerability requires admin panel access, a limited-privilege admin could exploit it
