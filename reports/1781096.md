## Program Summary

> The WARP client has a functionality that enables users to exclude or include IP Ranges from being proxied through the tunnel. The text field where said IP Ranges are entered did not properly validate the length of the IP Range inserted, resulting in the WARP client crashing if a long string was used instead of an IP Range.
