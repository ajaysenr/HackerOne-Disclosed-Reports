## Program Summary

> During a period of roughly week we had deployed a feature that would let you redact usernames in the Export PDF function that did not account for a number of edge cases including nullability, short names, and a few other edge cases. The function did not perform what it was aiming to do, and was reverted thereafter, during that time the hacker identified one of such edge cases and filed the below report.

## Vulnerability Details

**Summary:**

Hi Team,

Export report via .pdf still disclosing the reporter username despite I exported it with __"I want to redact all usernames"__ being selected to redact all the usernames.

{F2475645}

### Steps To Reproduce

1. Login and try to export any report using export as .pdf
2. Select "I want to redact all usernames"
3. Check the exported .pdf report is still disclosing the username of researcher
