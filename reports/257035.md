## Program Summary

> A security researcher reported an issue around user enumeration through examination of the failed registration error message. Since this change was reported, Legal Robot has switched to a method wherein any registration attempts for accounts that already exist will be redirected to the same pending-verification screen as successful registrations, which blocks this type of user enumeration.
