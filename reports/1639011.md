## Program Summary

> A security vulnerability has been identified where sensitive API keys are exposed through archived data accessible via the Wayback Machine (web.archive.org). Some of these API keys are found to be valid and could potentially be exploited by malicious actors. The exposure occurs through historical snapshots of Planet's API  endpoints that were cached by the Internet Archive's crawling service.
