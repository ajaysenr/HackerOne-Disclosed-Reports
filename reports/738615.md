## Program Summary

> ## Summary
An issue discovered in Yoti's marketing website (www.yoti.com) WordPress implementation. The vulnerability exposes admin usernames through WordPress API endpoints.

## Steps to Reproduce
1. Search Shodan using query `org:yoti.com`
2. Access the identified IP: `http://185.116.54.11`
3. Add the following parameters to access WordPress API endpoints:
   - `/wp-json`
   - `/wp-json/wp/v2/users`


## Impact
The vulnerability exposes admin usernames through WordPress API endpoints
