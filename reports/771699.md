## Program Summary

> ## Summary
An attacker can redirect a user to any external website using the vulnerable parameter in `https://account.mackeeper.com/auth/fb` use parameter `continue`.

## Steps To Reproduce
1. Visit the following url: `https://account.mackeeper.com/auth/fb?continue=https://google.com`
2. Login
3. This will redirect you to https://google.com
