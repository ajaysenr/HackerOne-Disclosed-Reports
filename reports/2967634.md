## Vulnerability Details

## Summary:
Proxy at https://52.90.28.77:30920 allows to access internal domains

## Steps To Reproduce:
To reproduce, simply use this curl command
  ```
curl --insecure https://52.90.28.77:30920/reddit --header "Host: █████████"
```
