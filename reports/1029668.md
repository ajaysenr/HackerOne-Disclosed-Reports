## Program Summary

> @tomorrow_future reported an issue where an unauthenticated user could reflect JavaScript on a product page for a very small number of shop configurations. This issue did not fall under our typical Storefront cross-site scripting exclusion because the input is provided by a visitor to the store, not an authenticated administrator.

## Vulnerability Details

1、install app `Product Reviews`
{F1070556} 

2、Open a product and write a review

3、Press F12 on the keyboard，Change the type of email to text.
