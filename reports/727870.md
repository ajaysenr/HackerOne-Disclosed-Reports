## Program Summary

> ## Summary
An issue identified in the WordPress installation at the marketing site www.yoti.com where user information of administrative accounts is publicly accessible. Through an accessible WordPress REST API endpoint, unauthorized users are able to retrieve sensitive information about administrator accounts. This exposure included usernames, creating a potential security risk for the WordPress installation.

## Steps to Reproduce
1. Access the following URL in a web browser:
```
https://www.yoti.com/wp-json/wp/v2/users
```
2. The response will display information about administrator users, including usernames and associated data
3. This information can be viewed without any authentication or special permissions

## Impact
The vulnerability allows attackers to obtain administrator usernames and associated information without authentication.
