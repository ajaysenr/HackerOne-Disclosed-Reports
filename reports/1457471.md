## Program Summary

> A privilege escalation issue in Shopify's Partner Portal (`partners.shopify.com`) allowed users without `View referrals` permission to create POS leads by directly accessing the lead creation URL. While the interface blocked unauthorized users from the referrals page, the backend API lacked proper authorization checks, enabling users to bypass restrictions and submit referrals without proper permissions.
