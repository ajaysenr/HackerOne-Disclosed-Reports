## Program Summary

> A CSRF (Cross Site Request Forgery) vulnerability was reported on TikTok which could potentially be used by an attacker to delete other users' public videos if the user were to click a malicious link. This issue has since been resolved. We thank @luizviana for reporting this to our team and confirming the resolution
